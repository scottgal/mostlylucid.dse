# article_writer_portable

Write and translate article - fully portable

## Requirements

- Python 3.7+
- Access to Ollama API endpoint (default: http://localhost:11434)

## Installation

```bash
pip install -r requirements.txt
```

## Usage

### Run with JSON input:

```bash
python run_workflow.py --input '{"topic": "Your Topic"}'
```

### Run with input file:

```bash
python run_workflow.py --input-file inputs.json
```

### Run with stdin:

```bash
echo '{"topic": "Your Topic"}' | python run_workflow.py
```

## Inputs

- `topic` (string) - **Required**\n  Article topic\n- `target_language` (string) - Optional (default: `Spanish`)\n  Language to translate to
